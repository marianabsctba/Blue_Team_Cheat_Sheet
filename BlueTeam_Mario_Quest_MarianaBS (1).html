📋 PLAYBOOKS & RUNBOOKS ▼

Playbook de Phishing

  1. Quarentena do email suspeito no gateway
  2. Extração de headers (origem, SPF/DKIM/DMARC)
  3. Análise de URLs (sandbox, VirusTotal, URLScan)
  4. Análise de anexos (sandbox, hashes, macros)
  5. Identificar usuários que clicaram/abriram
  6. Bloqueio de IoCs (URL, IP, domínio, hash)
  7. Verificar endpoint comprometido (EDR)
  8. Notificar usuários afetados
  9. Reset de credenciais se necessário
  10. Documentar e fechar ticket

Playbook de Malware Detectado

  1. Isolamento do endpoint (EDR quarantine/network isolation)
  2. Coleta de evidências (hash, processo, conexões)
  3. Análise do malware (sandbox, strings, imports)
  4. Identificação do vetor de entrada (email, web, USB)
  5. Busca por outros sistemas afetados (IOC sweep)
  6. Erradicação (remoção ou reimagem)
  7. Bloqueio de C2 nos controles de rede
  8. Restauração e monitoramento reforçado
  9. PIR (Post-Incident Review)

Playbook de Brute Force SSH/RDP

  1. Confirmar múltiplas falhas de login (threshold)
  2. Identificar IPs de origem
  3. Verificar se houve logon bem-sucedido posterior
  4. Bloquear IPs atacantes (firewall/WAF)
  5. Se comprometido: isolar, coletar evidências
  6. Reset de senha/credenciais
  7. Habilitar MFA se não existente
  8. Recomendar fail2ban/Account lockout policy

🎯 METODOLOGIA DE HUNTING ▼

Processo Threat Hunting

  1. Formular Hipótese — "Os atacantes usam T1059.001 em nosso ambiente"
  2. Identificar Fontes de Dados — Sysmon, EDR, DNS, proxy
  3. Criar Query — SPL, KQL, YARA, regex
  4. Executar e Analisar — Filtrar ruído, baseline
  5. Investigar Achados — Determinar malicious vs benign
  6. Documentar — Hunt report, IOCs encontrados
  7. Melhorar Detecção — Criar regra se encontrar algo

Hipóteses Populares de Hunting

  • PowerShell com encoding base64 suspeito
  • Processos com conexão externa inesperada
  • Scripts executados por Office Apps
  • Serviços criados em horário incomum
  • Usuários acessando dados nunca antes acessados
  • Beacon de rede com intervalos regulares (jitter <10%)
  • Living off the Land Binaries (LOLBins)
  • Kerberoasting — SPNs requisitados
  • DCSync attempt
  • Credenciais de admin usadas em múltiplos hosts

# Binários legítimos usados por atacantes certutil.exe → Download de payload bitsadmin.exe → Download, persistência mshta.exe → Executar HTA/scripts wscript.exe → VBS execution regsvr32.exe → Squiblydoo technique rundll32.exe → DLL execution powershell.exe → Tudo! 😱 wmic.exe → Lateral movement net.exe → Reconnaissance schtasks.exe → Persistência cmd.exe → Chaining commands
